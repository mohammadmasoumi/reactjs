## Installation

- Installing nodejs
  https://nodejs.org/en/

- Installing npm (node package manager)

## Create react app

- Install `create-react-app` package

```
# i: install
# g: global
npm i -g create-react-app
```

- Create app

```
# create-react-app APP_NAME
create-react-app react-app
cd react-app
npm start
```
